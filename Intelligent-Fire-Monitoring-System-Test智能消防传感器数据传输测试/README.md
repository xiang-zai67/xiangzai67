# 智能消防监测系统 - 测试项目

## 项目简介
本项目为**物联网毕业设计：智能消防监测系统**的配套测试工程，实现从物联网仿真实训平台采集传感器数据，上传至大陆物联网云平台（NLECloud），再通过Python脚本将数据同步到本地窗口化应用展示的全流程功能测试。

本人在项目中担任**测试负责人**，独立完成：
- 需求分析与测试点提取
- 测试用例设计与执行
- 数据链路功能/异常/流程测试
- 缺陷提交、跟踪与回归验证
- 测试总结输出

## 技术栈
- 测试类型：功能测试 / 流程测试 / 数据正确性测试 / 异常场景测试
- 开发语言：Python（数据同步脚本）
- 测试工具：手动功能测试、场景测试
- 业务场景：物联网、数据传输、消防监测

## 系统架构
物联网仿真实训平台（传感器数据） → 大陆物联网云平台（NLECloud） → Python 数据同步脚本 → 本地窗口化应用（展示/报警）

## 传感器列表
- 可燃气体（z_combustible）
- 温度（z_temperature）
- 湿度（z_humidity）
- 氧气（m_oxygen）
- 火焰（m_fire）
- 烟雾（m_smoke）

## 执行器与策略
- 报警灯（m_alarmlight）：火焰>0 或 可燃气体>50 时报警
- 逃生灯（m_lamp）：火焰>0 时开启
- 水泵（m_shuibeng）：火焰>0 时开启
- 风扇（m_fan）：烟雾>0 或 可燃气体>50 时开启

## 测试范围
1. 仿真平台数据上传云平台功能
2. 云平台数据接收与存储功能
3. Python 脚本数据拉取与转发功能
4. 本地应用数据显示与报警逻辑
5. 异常场景：断网、数据异常、重复上传、阈值触发

## 项目目录
