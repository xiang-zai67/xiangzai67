# 缺陷报告示例

## 缺陷1：云平台数据刷新延迟
- 模块：云平台数据展示
- 级别：一般
- 描述：数据上传后，云平台需3-5秒才刷新显示
- 复现步骤：上传数据 → 立即查看云平台设备传感器页面
- 预期：1秒内刷新显示最新数据
- 实际：3-5秒后才刷新
- 状态：已修复

## 缺陷2：极端温度数据显示异常
- 模块：本地应用UI
- 级别：轻微
- 描述：当温度超过100℃时，本地应用文本框显示格式错乱
- 复现步骤：设置温度=120℃ → 同步数据 → 查看本地应用
- 预期：正常显示数值，无格式错乱
- 实际：文本框内容重叠、乱码
- 状态：已修复

## 缺陷3：断网后脚本无提示
- 模块：Python数据同步脚本
- 级别：中等
- 描述：网络断开时，脚本直接崩溃，无友好提示
- 复现步骤：断开网络 → 点击“刷新数据”
- 预期：弹出网络错误提示，脚本继续运行
- 实际：脚本崩溃退出
- 状态：已修复
